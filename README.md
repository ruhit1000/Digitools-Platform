# DigiTools Platform

## Description
**DigiTools Platform** is a modern, responsive single-page web application for browsing and purchasing premium digital assets. Built with a user-first approach, the platform lets users explore tools, view product details, and manage selections through a dynamic shopping cart experience.

## Live Demo
[DigiTools Platform](https://digitools-platform-app.netlify.app/)

## Top 3 Features
1. **Interactive Shopping Cart**
Users can add items to cart, remove items, and instantly see updates in cart totals and the notification badge in the sticky navbar.

2. **Dynamic Tab Navigation**
A smooth sliding-pill toggle allows users to switch between the **Products** gallery and **Cart** summary without page reloads.

3. **Modern and Responsive UI**
Mobile-first design with Tailwind CSS and DaisyUI, including gradient accents, hover effects, and toast notifications for user feedback.

## Technologies Used
- **Frontend Framework:** React.js (Vite)
- **Styling:** Tailwind CSS v4, DaisyUI
- **State Management:** React `useState`
- **Notifications:** React-Toastify
- **Icons:** React Icons
- **Data Source:** Local JSON data

## How to Run Locally
1. **Clone the repository**
```bash
git clone https://github.com/ruhit1000/Digitools-Platform.git
```

2. **Navigate to the project directory**
```bash
cd digitools-platform
```

3. **Install dependencies**
```bash
npm install
```

4. **Start the development server**
```bash
npm run dev
```

5. **Open in your browser**
Use the local URL shown in the terminal (usually `http://localhost:5173`).

## Available Scripts
- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
